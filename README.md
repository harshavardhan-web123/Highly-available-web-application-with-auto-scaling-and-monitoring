# Highly-available-web-application-with-auto-scaling-and-monitoring

* Launch an EC2 instance and install Apache or Nginx web server.
* Deploy a Real time Web application.
* Create an AMI from the configured instance.
* Create an Application Load Balancer and configure a target group.
* Create a Launch Template using the AMI.
* Create an Auto Scaling Group with minimum 2 and maximum 4 instances.
* Attach the Load Balancer to the Auto Scaling Group.
* Configure scaling policies based on CPU utilization.
* Create CloudWatch alarms to monitor CPU usage.
* Create an SNS topic and subscribe using email to receive alerts
